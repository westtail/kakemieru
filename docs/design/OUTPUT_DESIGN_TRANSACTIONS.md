# アウトプット設計 - 明細モード

最終更新: 2026-04-07

---

## 概要

CSVから取り込んだ明細データを一覧表示・カテゴリ管理・分析するモード。

---

## 画面構成

```
/transactions
├── 絞り込みバー
├── 明細一覧テーブル
└── 分析パネル（タブ切替）
```

---

## 絞り込みバー

| 項目 | 内容 |
|------|------|
| 月 | 年月セレクト（デフォルト: 当月） |
| カテゴリ | セレクトボックス（共通＋ユーザー独自） |
| キーワード | 店舗名テキスト検索 |

---

## 明細一覧テーブル

| 列 | 内容 |
|----|------|
| 日付 | YYYY/MM/DD |
| 店舗名 | CSVの摘要・加盟店名 |
| 金額 | 円表示（カンマ区切り） |
| カテゴリ | セレクトで手動変更可能 |

- カテゴリ「未分類」の行は強調表示
- カテゴリ変更は即時保存（Turbo Stream）

---

## 分析パネル

明細一覧と同画面にタブ切替で表示。

### タブ1: カテゴリ別

- カテゴリ別円グラフ
- カテゴリ別金額一覧（金額・割合）

### タブ2: 月別推移

- 月別支出棒グラフ（直近12ヶ月）
- 前月比・前年同月比の表示

---

## カテゴリ管理

```
/categories
├── 初期カテゴリ一覧（category_templates からコピー済み・名前変更可）
└── ユーザー独自カテゴリ一覧（追加・編集・削除）
```

- 初期カテゴリ: 食費・交通費・娯楽・家賃・光熱費・通信費 など（`category_templates` からコピー）
- ユーザー独自カテゴリ: 自由に追加可能
- 全カテゴリは `categories.user_id NOT NULL`（ユーザーに必ず紐づく）

---

## 関連DBテーブル

| テーブル | 役割 |
|---------|------|
| `transactions` | 明細（CSVから取り込み） |
| `categories` | カテゴリ（user_id NOT NULL・コピー方式） |
| `category_templates` | システム共通テンプレート（登録時に categories にコピー） |
| `payment_methods` | 支払い手段（どのカード・QR等の明細か） |
